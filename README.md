# CastLink

CastLink is a React app for exploring shared connections across actors, movies, and TV shows using TMDB.

## What it does
- Compare two actors and see titles they share.
- Compare two titles and see shared cast.
- Run Bacon's Law to find the shortest movie-only path back to Kevin Bacon.

## Local development
```bash
npm install
npm run dev
```

## iOS (Capacitor)
```bash
npm run ios:sync
npm run ios:open
```

The iOS app wraps the built web app with Capacitor and is intended for TestFlight / App Store distribution.
